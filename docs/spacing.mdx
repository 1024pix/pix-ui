# Espacements

Pour uniformiser les espacements dans nos applications, on dispose de plusieurs variables SCSS :

- `$pix-spacing-xxs`
- `$pix-spacing-xs`
- `$pix-spacing-s`
- `$pix-spacing-m`
- `$pix-spacing-l`
- `$pix-spacing-xl`
- `$pix-spacing-xxl`
