# Pix Multi Select

Un select custom permettant une gestion de la multiselection. Permet de passer soit du texte brut soit des composants Ã  sa liste. (PixStars etc...)
l'ajout de \`class\` et d'autres attributs comme \`aria-label\` sont possibles.

## Usage

```js
<PixMultiSelect
  @title={{title}}
  @emptyMessage={{emptyMessage}}
  @id={{id}}
  @onSelect={{doSomething}}
  @selected={{selected}}
  @options={{options}} as |option|>
  {{option.label}}
</PixMultiSelect>
```
