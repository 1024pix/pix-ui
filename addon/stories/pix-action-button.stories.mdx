# Pix Action Button

Le ActionButton permet de créer un bouton contenant une icône font-awesome.

<Canvas>
  <Story name="Action button" story={stories.actionButton} height="60px" />
</Canvas>

## Usage

état enabled
```html
<PixActionButton @icon='times' @triggerAction="{{triggerAction}}"/>
```

état disabled
```html
<PixActionButton @icon='times' @triggerAction="{{triggerAction}}" disabled={{true}} aria-disabled="{{true}}"/>
```
