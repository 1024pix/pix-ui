# Pix Button

Ce composant est un bouton simple qui empêche les clics multiples.

<Canvas isColumn>
  <Story name="Disabled" story={stories.disabledButtons} height="60px" />
</Canvas>

## Usage

```html
Par défaut, le bouton est actif
<PixButton
  @isDisabled={{false}}
  >
  Cliquez pour valider !
</PixButton>

Bouton désativé
<PixButton
  @isDisabled={{true}}
  >
  Cliquez pour valider !
</PixButton>
```
