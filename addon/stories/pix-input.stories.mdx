# PixInput

Affiche un champ Input.

Peut recevoir des propriétés classiques d'un élément input,
car il hérite de `...attributes`.

Peut recevoir un paramètre optionnel `searchableOptions` (tableau d'objet contenant
les propriétés `value` et `label`) permettant de proposer à l'utilisateur une liste
d'option. Si l'utilisateur choisit une option valide, la bordure devient verte.
