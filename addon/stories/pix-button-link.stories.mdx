# Pix Button

Ce composant est un bouton simple qui empêche les clics multiples.

<Canvas isColumn>
  <Story name="Link" story={stories.linkButtons} height={200} />
</Canvas>

## Usage

```html
Par défaut, le composant est un bouton
<PixButton
  @isLink={{false}}
  >
  Cliquez pour valider !
</PixButton>

Composant LinkTo
<PixButton
  @isLink={{true}}
  @route='profile'
  >
  Cliquez pour valider !
</PixButton>
```
