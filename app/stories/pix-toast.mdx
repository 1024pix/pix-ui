# Pix Toast

Une notification avec icône et un bouton de fermeture.

## Usage

> **⚠️** **Pour utiliser `<PixToast />` sur l'application, utiliser `<PixToastContainer />`**

```html
<PixToastContainer />
```

Quatre styles de notification sont disponibles.

```js
@service pixToast;

this.pixToast.sendErrorNotification({
  message: 'Une erreur est survenue',
  ariaLabel: 'Fermer la notification'
});

this.pixToast.sendSuccessNotification();

this.pixToast.sendInformationNotification();

this.pixToast.sendWarningNotification();
```
